# 🤖 Atividade: Reconstrução de Software Assistida por IA (Clone ao MVP+)

## 📝 Descrição da Atividade
Nesta atividade, você assume a posição de **Desenvolvedor de Software assistido por IA**. O objetivo principal é reconstruir um aplicativo funcional a partir da pura observação de sua interface externa, sem visualizar o código-fonte original ou fornecer o link direto para a IA destrinchar.

---

## ⚙️ Ciclo de Desenvolvimento

O processo de engenharia reversa e desenvolvimento foi dividido em três etapas fundamentais:

* **Analisa:** Você acessa o link do webapp de referência, explora detalhadamente suas funcionalidades e mapeia todos os componentes visuais e regras de lógica de negócio presentes na interface.
* **Configura:** No Google AI Studio, você define as *System Instructions* do modelo (Gemini), especificando que ele deve atuar como um desenvolvedor Full-Stack. Você descreve a estrutura de arquivos necessária (HTML, CSS e JS) e o comportamento esperado para cada interação do usuário.
* **Constrói e Valida:** Você gera o aplicativo completo dentro do Google AI Studio. Você executa o código gerado no ambiente de teste, compara a funcionalidade com o link original e ajusta as instruções até que o software final apresente o mesmo comportamento e estética da referência.
